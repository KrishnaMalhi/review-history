# ReviewHistory — Business Model

## 💡 Core Principle

**Free until 10,000+ users.** Revenue kills growth if introduced too early. Trust first. Community first. Money second.

---

## 💰 Revenue Streams

### Stream 1: Verified Business Badge (Month 6+)

- **What:** Businesses pay to show a "Verified & Responsive Owner" badge
- **Price:** PKR 1,500/month (~$5.50)
- **Value to business:** Builds customer confidence, shows they care, stands out from unverified listings
- **Acquisition:** Notify top-rated entities (Trust Score 70+) via in-app message — no cold calling
- **Projections:**
  - Month 6: 30 businesses × PKR 1,500 = PKR 45,000/month
  - Month 9: 100 businesses × PKR 1,500 = PKR 150,000/month
  - Year 1: 300 businesses × PKR 1,500 = PKR 450,000/month

### Stream 2: Featured Listing (Month 6+)

- **What:** Top-rated entity appears first in search results for their category + city
- **Price:** PKR 2,000/month per category per city
- **Example:** "Top Doctor in Johar Town, Lahore" slot
- **Projections:**
  - Month 6: 20 listings × PKR 2,000 = PKR 40,000/month
  - Year 1: 100 listings × PKR 2,000 = PKR 200,000/month

### Stream 3: Reputation Dashboard (Year 1+)

- **What:** Analytics for claimed businesses — review trends, sentiment, competitor comparison
- **Price:** PKR 3,000/month
- **Target:** Doctors, property agencies, clinics, mid-size shops
- **Projections:**
  - Year 1: 50 businesses × PKR 3,000 = PKR 150,000/month
  - Year 2: 200 businesses × PKR 3,000 = PKR 600,000/month

### Stream 4: B2B API / Data Licensing (Year 2+)

- **What:** Property portals embed ReviewHistory trust scores in their listings
- **Potential Partners:** Zameen.com, OLX Pakistan, Rozee.pk, Daraz
- **Price:** Custom enterprise contracts
- **Projection:** PKR 500,000+/month per major partner

### Stream 5: Premium User Plan (Year 1+)

- **What:** Power users pay for advanced features
- **Price:** PKR 299/month
- **Features:** Notifications for new reviews on saved entities, export data as PDF, see review analytics, priority support
- **Projection:** 500 users × PKR 299 = PKR 149,500/month

## 💳 Payment Methods (Pakistan)

| Method | Use Case |
|--------|----------|
| JazzCash | Most popular mobile wallet in Pakistan |
| EasyPaisa | Second largest mobile wallet |
| Bank transfer | Enterprise B2B contracts |
| Stripe | International card payments |

### JazzCash Integration Flow

```
1. User selects JazzCash on checkout
2. Enter JazzCash mobile number
3. Receive USSD push notification
4. Confirm payment with JazzCash PIN
5. Webhook confirms payment to ReviewHistory API
6. Badge/subscription activated immediately
```

### EasyPaisa Integration Flow

```
1. User selects EasyPaisa on checkout
2. Redirected to EasyPaisa payment page
3. OTP sent to registered number
4. OTP confirmed → payment processed
5. Callback URL notifies ReviewHistory API
6. Subscription activated
```
